运行环境：
建议使用 bochs 虚拟机，下载地址： https://sourceforge.net/projects/bochs/files/bochs/2.3.5/Bochs-2.3.5.exe/download

汇编编译器：NASM

使用时clone本项目，解压 80m.img.bak.gz 得到镜像文件 80m.img

详细使用方法待补充...

Note:
1. kliba.asm and klib.c will be compiled to distinct bin
   kliba.asm -> kliba.o
   klib.c -> klib.o
